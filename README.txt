1. Восстановить таблицы из countries.sql (страны) и init.sql (выставочные)  - там страны и таблица посетителей, она пустая.
2. Чтобы денверский mysql работал с utf-8, в my.cnf (ДИСК:\usr\local\mysql-5.1)от денвера изменить параметры:
    [client]
    ...
    default-character-set = utf8
    ...
    [mysqld]
    ...
    default-character-set = utf8
    init-connect = "set names utf8"
    
2.1 Иногда нужно поправить файл php.ini(Z:\usr\local\php5)

   default_charset = "utf8"
   
3. Настройки изменяются через файл settings.php
   badges - параметры бейджей (размеры, положение текста, нужно ли печатать фон и т.п.)
   db - параметры базы данных 
   admin - параметры админки (пароль)
   
4. Админка: в браузере в auth.php ввести пароль (123456 по умолчанию), появится ссылка на админку

5. В админке доступна загрузка-выгрузка посетителей в формате csv:
 
   barcode - ид посетителя, номер на штрихкоде, 
   firstname, lastname, patronymic - ФИО,
   post, company, - должность и компания,
   country, town, 
   address, postcode - адрес
   email, website 
   exibition_id, 
   preregister, 
   visited

6. Формат описания файла с бейджем:
{   //layout - Схема расположения текста на бейдже
    "layout" : {
        //Параметры для куска текста
        "normal" : {
            //Поля из таблицы visitor, сгруппированые по строкам
            "fields" : [
            ["lastname", "firstname", "patronymic"],
            ["post", "company"]
            ],
            //Ограничение размера шрифта
            "fontsize" : 40,
            //Границы области, относительно которой выравнивается текст
            "bounds" : [450, 1540, 1900, 1890],
            //Отступы от краёв области для текста
            "padding" :[100, 100, 100, 100],
           //Расстояние между строками текста
            "interval":30,
            //Поворот, 0, или 180 градусов
            "angle" : 0
        }
    },
    //Путь к шрифту, которым рисуется текст
    "font" : "img/fonts/LiberationSans-Regular.ttf",
    //Параметры штрихкода
    "barcode" : {
         //Координаты левого верхнего угла
        "position" : [800, 2150],
        //Высота
        "height" : 150,
        //Ширина полоски
        "barwidth" : 8,
         //Размер шрифта, которым написан номер
        "fontsize" : 30
    },
    //Путь к картинке с бейджиком
    "template" : "img/badges/plastic.png",
    //Нужно ли рисовать фон
    "background" : true,
    //Тип картинки на выходе
    "type" : "image/png"
}

7. Импорт:
   csv в кодировке cp1251, разделитель полей - ;
   
8. Пул штрихкодов отключается параметром pool_disabled в settings.php
